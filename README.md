## ✨ 功能特性

### 🎯 核心计时功能

- **自定义学习时长**：支持设置任意学习时间
- **倒计时/正计时模式**：灵活选择计时方式
- **课间休息**：一键暂停，休息时间计时显示
- **休息提醒**：可自定义提醒时间（默认50分钟）

### 🎵 环境音效系统

- **多种环境音效**：篝火雨夜、森林草原、风声心跳、冷气声
- **音效视觉联动**：
  - 篝火粒子效果（计时框内跳动）
  - 雨滴划过效果（前景玻璃圆片）
  - 闪电效果（随机间隔闪烁）
- **音量控制**：支持自定义音量并记忆设置

### 🎨 视觉效果

- **视差背景**：多层元素跟随鼠标移动产生深度感
- **背景切换**：学习模式/休息模式自动切换背景
- **平滑过渡**：背景切换带有淡入淡出动画
- **毛玻璃效果**：前景按钮和计时器采用玻璃态设计

### 📊 数据统计

- **学习记录**：自动记录每次学习时长
- **数据可视化**：
  - 近6天学习时长柱状图
  - 全部天数学习时长折线图
  - 鼠标悬停显示详细信息
- **数据导入/导出**：支持 JSON 格式备份

### ⚙️ 设置功能

- **休息提醒时间**：自定义提醒间隔
- **屏幕常亮**：可设置自动激活间隔防止息屏
- **音量记忆**：自动保存上次音量设置

### 💾 数据持久化

- **本地存储**：学习记录和设置自动保存到浏览器 localStorage
- **状态恢复**：关闭页面后重新打开可恢复学习状态

## 🎮 使用说明

### 基本操作

1. **开始学习**：点击「开始自习」按钮，设置学习时长后开始
2. **课间休息**：点击「课间休息」暂停学习
3. **结束自习**：点击「结束自习」保存记录并退出
4. **环境音**：点击「环境音」按钮选择背景音效
5. **设置**：点击「设置」按钮调整各项参数

## 🔧 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript (ES6+)**：逻辑实现
- **Canvas API**：数据可视化图表
- **Web Audio API**：音频播放控制
- **LocalStorage API**：数据持久化
- **Wake Lock API**：屏幕常亮功能
